Featured

Distill: visual, interactive machine learning journal
colah.github.io

“Machine Learning Research Should Be Clear, Dynamic and Vivid.” - so say hello to Distill, a visual interactive journal for machine learning research to be presented in a way that is intended for human understanding. With so many ML articles contributing to the pile of ‘research debt’ (produced when people rush out new discoveries without putting effort into communication), Distill is positioning as an antidote to that. Clarity is the goal, and it does a great job at providing visualisations that can really help build intuitive understandings of what techniques do, and what’s appropriate where. Bravo Team Distill!
